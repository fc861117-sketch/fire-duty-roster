# Fire Duty Roster

消防分隊勤務自動排班網站。輸入前一日深夜勤人員與今日上班人員後，可自動生成勤務表，並支援其他勤務欄位、手動調整、規則校驗、CSV 匯出與 JSON 備份。

## 功能

- 前一日深夜勤人員自動安排 08-10 休息。
- 一天最多同時 2 位主管上班；若當日有 2 位主管，其中 1 位需由你填入外宿欄位。
- 主管以值班勤務為主，接下來才是救護勤務；主管避開深夜勤務，值班時段原則為 08-10 與 20-22。
- 主管原則上不編排救護1車。
- 救護1車全日編排。
- 救護2車依當日上班人數（不含役男）編排：5 人以下全日不編排；6 人時編排 08-12、14-18；7 人以上全日編排；若因其他勤務導致人數不足可不編排。
- 除主管及役男外，每人救護1/2車勤務量盡可能相近。
- 救護1車排班原則 4 小時一段，得依人力或勤務調整為 2 小時。
- 救護2車原則 2 小時換班一次，不可連續超過 4 小時。
- 00-08 救護勤務人員不必換人，不受 4 小時上限限制。
- 救護勤務須編排 2 人，不得兩女同車；主管不得同時編排同車救護勤務。
- 20-00 救護2車安排深夜救護1車人員。
- 24 小時值班欄至少需有 1 人。
- 值宿勤務為 22-06，需由同一人編排。
- 每日 16-17 備勤人員排入「訓練」。
- 休息班精確 2 小時，避開 16-17 與 22-06。
- 外宿人員預設休息時段為 18-20。
- 外宿同仁 20-08 不會出現在勤務表上，包含值班、救護、備勤與其他勤務。
- 值班優先順序為役男、主管、隊員。
- 役男預設 19,20，可彈性調整值班時段；每日值班上限 10 小時，若當日有 2 位役男上班，則每人原則 8 小時，且不可排 22-06 值宿。
- 新人每日需由資深同仁帶領訓練至少 2 小時，資深同仁不含主管與役男。
- 主表欄位順序為：時段 \ 項目、值班、救護1車、救護2車、消防查察、會勘/開會、防災宣導、訓練、備勤、其他、休息。
- Google 試算表只讀載入：在網站內顯示每月每人深夜勤次數、平均、最高/最低、差距與下次優先建議，不會編輯雲端試算表。

## 使用

直接開啟 `index.html` 即可使用。

也可用任一靜態網站服務部署，例如 GitHub Pages。

## 檢查

可用 Node.js 執行 `node test-roster-rules.js`，會自動生成常見情境並檢查外宿、值班與救護硬性規則。

## Google 試算表統計

在「深夜勤平均統計」區塊貼上 Google Sheets 連結後讀取資料，確認日期欄、人員欄、勤務欄與月份，即可產生統計。

若試算表不是公開或已發布資料，瀏覽器可能無法直接讀取；建議使用「發布到網路」的 CSV 連結，或將試算表分享權限設為知道連結者可檢視。
